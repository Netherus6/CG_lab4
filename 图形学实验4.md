## 三、 实验原理
### 1. Phong光照模型
Phong模型将表面某点的光照颜色分为三个部分计算：  
[  
I = I_{ambient} + I_{diffuse} + I_{specular}  
]

+ **环境光**：( I_{ambient} = K_a \cdot I_{light} \cdot C_{obj} )
+ **漫反射**：( I_{diffuse} = K_d \cdot (N \cdot L) \cdot I_{light} \cdot C_{obj} )
+ **镜面高光**：( I_{specular} = K_s \cdot (R \cdot V)^{N_s} \cdot I_{light} )

其中：

+ ( N )：表面法线；( L )：入射光方向；( V )：观察方向；( R )：反射光方向。
+ ( K_a, K_d, K_s )：材质对环境光、漫反射和镜面反射的反射系数。
+ ( N_s )：高光指数（Shininess），值越大高光范围越小越亮。

### 2. 光线-物体求交
+ **球体**：解一元二次方程 ( |O + tD - C|<sup>2 = r</sup>2 )。
+ **圆锥**：通过坐标变换和圆锥曲面方程求解，并需判断交点是否在高度范围内。

## 六、 实验心得与总结
1. **Phong模型的理解**：环境光保证物体不完全是黑色；漫反射决定物体色调和明暗过渡；高光决定材质的光泽感，三者缺一不可。
2. **Taichi框架的优势**：通过 `@ti.kernel` 自动并行化，代码逻辑接近普通Python循环，但运行在GPU上，性能极高，非常适合光线追踪等像素级独立计算的任务。
3. **几何求交的难点**：
    - 圆锥求交需要处理局部坐标系转换和高度约束，比球体复杂。
    - 法线计算必须归一化，否则光照会出现异常条纹。
4. **调试技巧**：
    - 当出现非预期黑块时，首先检查法线方向是否指向光线入射方向。
    - 可以通过临时将颜色设为法线颜色( `color = (N+1)/2` )来可视化法线是否正确。
5. **改进方向**：
    - 增加阴影计算（在光源方向发射阴影射线）。
    - 添加更多几何体（立方体、圆环）并实现反射/折射效果。
    - 引入纹理映射，实现更复杂的表面细节。

```python
# Phong光照计算核心
L = normalize(light_pos - p)
V = normalize(ro - p)

ambient = Ka[None] * light_color * hit_color

diff = ti.max(0.0, N.dot(L))
diffuse = Kd[None] * diff * light_color * hit_color

R = normalize(reflect(-L, N))
spec = ti.max(0.0, R.dot(V)) ** shininess[None]
specular = Ks[None] * spec * light_color 

color = ambient + diffuse + specular
```
